Project Bureau

Project Bureau aims at documenting the bitstream format of Microchip ATF15xx CPLDs. This is a work in progress. See more in the repository.

This HTML documentation has been automatically generated from a structured chip database. The complete chip database can be downloaded as a JSON file.
